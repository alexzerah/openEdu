# Langage compilé et interprété

## Langage compilé

Le langage compilé est d'abord compilé en langage machine, puis il est executé.

Les performances sont en général meilleure et les erreurs sont detectés en amont.

Ex : 
- C
- C++
- Rust
- Go

## Langage interprété

Le langage est exécuté au moment de l'exceution,

Pour cela, il a besoin d'un runtime.
Ces langages sont dit "moins performants"

Ex :
- Python
- JavaScript
- Ruby
