# CSS Grid

Afin d'utiliser grid, il est nécessaire d'avoir un parent (wrapper) qui contient les éléments de la grille.

Ce parent aura pour propriété `display: grid;`

Ce parent aura aussi la propriété
`grid`, qui 
grid-template-rows
grid-template-columns
grid-template-areas
grid-auto-rows
grid-auto-columns
grid-auto-flow
