# C0 - Hello world

## Hello world

```c
// main.c
#include <stdlib.h>

int main(void)
{
	write(1, "hello world\n", 12);
	return (0);
}
```

```shell
# Dans le terminal
cc -Wall -Werror -Wextra main.c -o helloWorld && ./helloWorld
```

Que se passe-t-il ? 

- le fichier `main.c` est compile et devient un programme `helloWorld`.
- La fonction d'entree `main` qui ne prend aucun parametre d'entree (`void`) et retoure un `int`.
- Elle affiche dans la sortie standard (`1`) la chaine de (12) caractère, ici `"hello world"` avec un retour à la ligne `\n`.
- L'usage de la fonction `write` est permis grace au header `<stdlib.h>`
- L'entier `int` `0` est renvoyé au processus pour indiquer que tout s'est bien passé (`1` dans le cas contraire).
