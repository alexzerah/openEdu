# C3 - Chaînes de caractères (char[])

## Définition

Une chaîne de caractères en C est :

un tableau de char terminé par un caractère spécial : `'\0'`

## Exemple

```c
char str[] = "Hello";
```

```bash
H   e   l   l   o   \0
72 101 108 108 111   0
```

Le `'\0'` marque la fin de la chaîne

```c
#include <stdio.h>

int main(void)
{
	char str[] = "Hello world";
	printf("%s\n", str);
	return (0);
}
```

%s signifie : “affiche une chaîne jusqu’à rencontrer '\0'”

contrairement à `write()`, pas besoin de donner la taille
