# C - Cours 2

## Types

Un type définit :

- la taille en mémoire
- la nature des données

Exemples :

- `char` → 1 octet
- `int` → généralement 4 octets (dépend de la machine)

## Le type char

char représente 1 octet.

Il peut contenir :

- un caractère ('A')
- ou une valeur numérique

```c
#include <unistd.h>

int main(void)
{
	char c = 'A';
	write(1, &c, 1);
	return 0;
}
```

Explication

- La variable `c` contient 'A'
- `&c` → adresse mémoire de `c`
- 1 → on lit 1 octet

> `char` un octet interprété comme un caractère

## Le type int

Définition
int représente un nombre entier.

taille typique :
- 4 octets (32 bits)

```c
char c = 'A';
int x = 10;
```

En mémoire (simplifié)

```bash
c → [65]
x → [0][0][0][10]
```

```shell
man ascii
```

```c
char a = 65;
write(1, &a, 1);

char a = 'A';
write(1, &a, 1);
```

## Introduction à printf

Jusqu’ici :

- on manipule des octets bruts avec write

Problème :

- difficile d’afficher des nombres
- difficile d’afficher du texte sans compter

```c
#include <stdio.h>

int main(void)
{
	int x = 10;
	printf("%d\n", x);
	return 0;
}
```

- `%d` : entier (int)
- `%c` : caractère (char)
- `%s` : chaîne de caractères
