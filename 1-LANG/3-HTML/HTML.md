# HTML

## Définition

- HTML (Hypertext Markup Language) est un "langage" web.
- C’est un langage de balisage.
- HTML5

## Balises

`<head>`

Balise contenant les informations annexes du site (titre, lien css...)
Par exemple
- `<meta>`
- `<title>`

## Ecrire du HTML

`index.html`

```
<html>
<head>
</head>
<body>
</body>
</html>
```
