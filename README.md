# openEdu

openEdu est un projet open source dédié à la formation en science informatique.

L’objectif est de fournir :
- Un cadre
- Des outils
- Des cours

pour aider 

- Les enseignants
- Les étudiants
- Les développeurs
- Les institutions

Un peu comme un SDK ou framework pour l’éducation tech.

## Objectifs

- Cours informatiques
- Guides pratiques pour concevoir des projets
- Promouvoir les bonnes pratiques de développement (architecture, documentation, modularité).

## Contributions

Les contributions sont encouragées. openEdu est un projet open source.
